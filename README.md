## About

This is my personal portfolio website, built from scratch with vanilla HTML, CSS and JavaScript — no frameworks, no dependencies, no build tools. Just clean, fast, static files deployable anywhere.

The design language is intentionally tech-forward: dark navy background, cyan blue accents, custom cursor, scroll-reveal animations, and a subtle grid overlay — reflecting the intersection of computational science and modern software engineering that defines my work.

## Pages

| File | Description |

## Tech Stack

- **HTML5 / CSS3 / Vanilla JS** — no frameworks
- **Google Fonts** — Syne, DM Sans, Space Mono
- **CSS custom properties** — full design token system
- **IntersectionObserver API** — scroll reveal animations
- **GitHub Pages** — free static hosting

---

## Features

- ⚡ Zero dependencies — pure HTML/CSS/JS
- 🎨 Custom animated cursor with ring follow effect
- 📜 Scroll reveal animations with directional entry (left/right/bottom)
- 🔵 Consistent dark blue design system across all pages
- 📱 Fully responsive — mobile, tablet, desktop
- 🔗 DOI-linked scientific publications — click images to open papers
- 🟡 Dual visual language — blue for apps, amber for scientific code tools
- 🏗️ Static — deploys instantly on GitHub Pages, Netlify, or Vercel

---

## Deployment

Hosted on **GitHub Pages** — automatically served from the `main` branch.

To run locally, just open `index.html` in any browser — no server needed.
